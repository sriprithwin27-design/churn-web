# Customer Churn Prediction & Visualisation

## Project Overview

This project is an end-to-end machine learning web application designed to predict customer churn in a telecom environment. The system combines data preprocessing, machine learning modelling, Flask backend development, and a user-facing prediction interface to simulate a real-world business analytics solution.

The application allows users to input customer behavioural and subscription-related parameters and receive:
- churn prediction results
- model interpretation
- business-oriented retention recommendations

The project was designed not only as a machine learning exercise, but as a practical analytics product focused on business decision-making and customer retention strategy.

---

# Business Problem

Customer churn is one of the most critical challenges faced by subscription-based businesses such as telecom providers. Acquiring new customers is significantly more expensive than retaining existing ones.

The objective of this project is to:
- identify customers likely to churn
- provide interpretable insights
- suggest actionable retention strategies
- simulate how predictive analytics supports business operations

---

# Features

- End-to-end machine learning workflow
- Logistic Regression churn prediction model
- Flask-based backend application
- Interactive web interface
- Dynamic prediction result page
- Business recommendations based on churn risk
